# Установка Golos

**Что нужно:** Mac на Apple Silicon (чип M1 или новее) и macOS 13 или новее.

> Как проверить чип: меню Apple () → «Об этом Mac» → строка «Чип» должна быть
> Apple M1/M2/M3/… Если там Intel — увы, приложение не запустится.

## 1. Установить

Открой файл `golos.dmg` двойным кликом и перетащи иконку **golos** в папку **Программы**.

## 2. Первый запуск (важно — иначе macOS заблокирует)

Приложение не подписано через Apple, поэтому при первом открытии система ругнётся.
Это ожидаемо. Чтобы разрешить:

1. Открой **Терминал** (Spotlight: ⌘+Пробел → набери «Терминал» → Enter).
2. Вставь эту строку и нажми Enter:
   ```
   xattr -dr com.apple.quarantine /Applications/golos.app
   ```
3. После этого открывай golos обычным двойным кликом из Программ.

## 3. Выдать разрешения

При первом запуске Golos попросит доступы — нужно разрешить все:

- **Микрофон** — чтобы слышать речь.
- **Мониторинг ввода** и **Универсальный доступ** (Настройки → Конфиденциальность
  и безопасность) — чтобы работала горячая клавиша и вставка текста.

Если случайно отказал — включи вручную в **Системные настройки → Конфиденциальность
и безопасность**, найди там golos и поставь галочку.

## 4. Первый раз — скачается модель распознавания

После первого запуска приложение само загрузит модель распознавания речи
(это разово, нужен интернет, займёт минуту). Дождись окончания.

## 5. Как пользоваться

Зажми клавишу **Right Option** (правый Alt) → говори → отпусти. Распознанный
текст вставится в то поле, где стоит курсор.
